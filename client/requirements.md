## Notes
Google Maps API Key is stored in localStorage ('google_maps_api_key').
Script injection for Google Maps is handled dynamically.
Dark mode is the default.
Primary color is Green (#22c55e).
Persistence: Stops and startAddress are saved to localStorage.
